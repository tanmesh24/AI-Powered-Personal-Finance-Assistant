# AI-Powered Personal Finance Assistant
A full-stack web application to manage finances, with AI-driven insights, budget tracking, and gamified features. Built with Java/Spring Boot, React, PostgreSQL, and Weka for a Google interview.

## Features
- Transaction management with mock Plaid API
- Budget tracking and alerts
- AI-driven overspending predictions
- Gamified UI with badges
- Real-time notifications

## Tech Stack
- Backend: Java, Spring Boot
- Frontend: React
- Database: PostgreSQL
- ML: Weka
